CN# Face Mask Detection System

A CNN-based Face Mask Detection System built using TensorFlow, Keras, and OpenCV.

## Project Overview

This project classifies facial images into two categories:

* With Mask
* Without Mask

The model was trained on a dataset containing 7,553 images and achieved approximately 95.5% test accuracy.

## Technologies Used

* Python
* TensorFlow
* Keras
* OpenCV
* NumPy
* Matplotlib

## Results

* Dataset Size: 7,553 images
* Image Resolution: 128 × 128
* CNN Architecture: 3 Convolutional Layers
* Test Accuracy: 95.5%

## Files

* face_mask_detection_colab.ipynb — Training notebook
* mask_detector.h5 — Trained model
